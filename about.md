
## Tentang SheetBlog

SheetBlog adalah eksperimen arsitektur web modern yang menggabungkan kemudahan penggunaan **Google Sheets** sebagai database dengan performa **React** yang responsif.

### Kenapa Memilih Kami?
1. **No-Build Architecture**: Website ini berjalan langsung di browser tanpa perlu server kompilasi yang rumit.
2. **Super Fast**: Menggunakan caching cerdas untuk memastikan konten dimuat seketika.
3. **Fully Customizable**: Mendukung dark mode dan 8 palet warna tema yang bisa diganti secara real-time.

### Teknologi yang Digunakan
- **React 19**: Library UI paling populer.
- **Tailwind CSS**: Framework styling modern untuk desain yang bersih.
- **Babel Standalone**: Memungkinkan penulisan JSX langsung di file HTML.
- **Google Sheets API (CSV)**: Database gratis dan mudah dikelola oleh siapa saja.
